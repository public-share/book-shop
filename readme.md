#### 20170112需要实现以下功能
1. 前台用户注册
2. 管理后台中可以查看已经注册的用户的信息

```
用户注册需要填写的个人信息如下
1. 手机号,作为登录账号使用(mobile)
2. 密码(pwd)
3. 邮箱(email)
4. 姓名(name)
5. 收件地址(address)
6. 注册时间(reg_time,需要你设计数据结构的时候填写默认值)
用户信息的数据集合为 users
用户管理后台的访问地址为:
	列表	/admin/users/list/1
	新增	/admin/users/add
	修改	/admin/users/edit
用户前台注册页面的访问地址为:
	/reg
前台页面要求使用weui实现手机页面效果
```
